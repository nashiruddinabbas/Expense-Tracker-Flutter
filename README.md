# Expenses Tracker App

A Daily Expenses Tracker App built using Flutter, Bloc, and Firebase.

## 🚀 Features

- **Beautiful Design**: Crafted with attention to detail using modern UI packages.
- **State Management**: Utilizes Bloc for efficient, clean, and manageable state handling.
- **Backend Integration**: Powered by Firebase for robust and scalable backend functionality (Authentication, Database).

## 🛠️ Technology Stack & Environment

- **Flutter**: Version 3.38.9 (Stable Channel)
- **Dart**: Version 3.10.8
- **Android SDK**: Target SDK 34 (Android 14)
- **State Management**: Bloc / Flutter Bloc
- **Backend**: Firebase Core & Firestore

## 📦 Packages Used

- `fontawesome_flutter`: For beautiful icons.
- `fl_chart`: For stunning charts and graphs.
- `intl`: For internationalization and date/number localization.
- `flutter_colorpicker`: For providing color picker widgets.
- `firebase_core` & `cloud_firestore`: For initializing and managing Firebase database.
- `firebase_auth`: For secure user authentication.
- `bloc` & `flutter_bloc`: For implementing the Bloc pattern.
- `equatable`: For simplifying object equality comparisons.
- `uuid`: For generating unique identifiers.
- `expense_repository`: Custom package for managing expense data models.
- `redacted`: Automatically generate beautiful skeleton loading widgets.
